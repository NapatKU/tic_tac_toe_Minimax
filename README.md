# Tic-Tac-Toe with Minimax Algorithm

This project is a **Tic-Tac-Toe game** where the computer opponent uses the **Minimax algorithm** to make optimal decisions.  
The AI is designed to always choose the best available move, making it impossible for the human player to win if both play perfectly.

---

## Features
- Human vs AI Tic-Tac-Toe gameplay
- AI uses **Minimax algorithm** with full game-state evaluation
- Guarantees **optimal play** from the computer
